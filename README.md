# 🌱 AgriLink

AgriLink is a MERN stack-based web application that connects farmers directly with buyers, eliminating middlemen and ensuring fair pricing and transparency in agricultural trade.

🚀 Features
🌾 Direct farmer-to-buyer marketplace
📦 Product listing and management
🔐 User authentication (login/signup)
💰 Transparent pricing system
📊 Scalable backend using REST APIs

## 🛠️ Tech Stack
Frontend: React.js, HTML, CSS, JavaScript
Backend:Node.js, Express.js
Database: MongoDB

## 📁 Project Structure
agri-link/
│── agrilink-backend/     # Backend APIs
│── agrilink-frontend/    # React frontend
│── frontend/             # Static files

## ⚙️ Installation & Setup
### 1️⃣ Clone the repository
git clone https://github.com/Jyoti-yadav4/Agri-link.git
### 2️⃣ Backend setup
cd agrilink-backend
npm install
npm start

3️⃣ Frontend setup
cd agrilink-frontend
npm install
npm start
